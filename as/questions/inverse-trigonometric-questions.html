INVERSE TRIGNOMETRIC FUNCTION QUESTIONS


Question 1:Let f : (-1,1) \(\rightarrow\) B, be a funcion defined by f(x) = \(tan^{-1}({2x\over {1-x^2}})\) , then f is both one-one and onto when B is the interval-

Answer : (-\(\pi\over 2\), \(\pi\over 2\))

Question 2: If \(cos^{-1}x\) - \(cos^{-1}{y\over 2}\) = \(\alpha\) then \(4x^2\) - 4xycos\(\alpha\) + \(y^2\) is equal to -

Answer : \(4sin^2\alpha\)

Question 3: If \(sin^{-1}({x\over 5})\) + \(cosec^{-1}({5\over 4})\) = \(\pi\over 2\), then a value of x is -

Answer : 3

Question 4: The value of cot(\(cosec^{-1}({5\over 3})\) + \(tan^{-1}({2\over 3})\)) is equal to-

Answer : \(6\over 17\)

Question 5: If x, y, z are in A.P. and \(tan^{-1}x\), \(tan^{-1}y\) and \(tan^{-1}z\) are also in A.P., then

Answer : x = y = z

Question 6: Let \(tan^{-1}y\) = \(tan^{-1}x\) + \(tan^{-1}({2x\over {1-x^2}})\), where |x| < \(1\over{\sqrt{3}}\). Then the value of y is :

Answer : \(3x-x^3\over {1-3x^2}\)

Question 7: Find the sum of the series :

\(cot^{-1}7\) + \(cot^{-1}13\) + \(cot^{-1}21\) + \(cot^{-1}31\) + .......... to n terms

Answer : \(cot^{-1}{2n + 5\over n}\)

Question 8: Solve :

\(sin^{-1}x\) + \(sin^{-1}2x\) = \(\pi\over 3\)

Answer : \(1\over 2\)\({\sqrt{3\over 7}}\)

Question 9: Solve :

\(tan^{-1}(x-1)\) + \(tan^{-1}x\) + \(tan^{-1}(x+1)\)

Answer : \(tan^{-1}3x\)

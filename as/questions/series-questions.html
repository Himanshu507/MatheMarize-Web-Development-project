Question 1: If \(a_1\), \(a_2\), \(a_3\),...., \(a_n\),..... are in A.P. such that \(a_4\) - \(a_7\) + \(a_{10}\) = m, then find the sum of first 13 terms of A.P.

Answer : 13m

Question 2: Let If \(a_1\), \(a_2\), \(a_3\),.... be an A.P. such that \(a_1 + a_2 + .... + a_p\over a_1 + a_2 + .... + a_q\) = \(p^3\over q^3\) ; p \(\ne\) q. Then \(a_6\over a_{21}\) is equal to

Answer : \(121\over 1681\)

Question 3: Given the sum of the first n terms of an A.P. is 2n + 3\(n^2\). Another A.P. is formed with the same first term and double of the common difference, the sum of n terms of the new A.P. is :

Answer : 6\(n^2\) - n

Question 4: If a, b, c are in A.P. then \({(a - c)}^2\) equals

Answer : 4(\(b^2\) - ac)

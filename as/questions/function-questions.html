FUNCTION QUESTIONS

Question 1: A real valued function f(x) satisfies the function equation f(x - y) = f(x)f(y) - f(a - x)f(a + y) where a is given constant and f(0) = 1, f(2a - x) is equal to

Answer : -f(x)

Question 2: If x is real, the maximum value of \(3x^2 + 9x + 17\over {3x^2 + 9x + 7}\) is-

Answer : 41

Question 3: The largest interval lying in \(({-\pi\over 2}, {\pi\over 2})\) for which the function [ f(x) = \(4^{(-x)^2} \) + \(cos^{-1}({x\over 2} - 1)\) + log(cosx) ] is defined, is

Answer : [0, \(\pi\over 2\))

Question 4: Let f : N \(\rightarrow\) Y be a function defined as f(x) = 4x + 3 where Y = { y \(\in\) N : y = 4x + 3 for some x \(\in\) N }. So that f is invertible and its inverse is

Answer : g(y) = \(y - 3\over 4\)

Answer : f is one-one and onto R

Question 6: The domain of the function f(x) = \(1\over {\sqrt{|x| - x}}\)

Answer : \((-\infty, 0)\)

Question 7: If a \(\in\) R and the equation \(-3{(x - [x])}^2\) + 2(x - [x]) + \(a^2\) = 0 (where [x] denotes the greatest integer \(\le\) x) has no integral solution, then all possible values of a lie in the interval :

Answer : (-1, 0) \(\cup\) (0, 1)

Question 8: Suppose f(x) = sinx and g(x) = 1 - \(\sqrt{x}\). Then find the domain and range of the following functions : (a) fog     (b) gof

Answer :(a) domain is x \(\ge\) 0; range [-1, 1]

(b) domain is 2k\(\pi\) \(\le\) x \(\le\) 2k\(\pi\) + \(\pi\); range [0,1]

Question 9: Find the inverse of the function f(x) = \(ln(x + \sqrt{x^2 + 1})\)

Answer : \(e^x - e^{-x}\over 2\)

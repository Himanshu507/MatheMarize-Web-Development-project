DIFFERENTIABILITY

Question 1: Suppose f(x) is differentiable at x = 1; \(\displaystyle{\lim_{x \to 0}}\) \(1\over h\)f(1+h) = 5, then f'(1) equals-

Answer : 5

Question 2: If f is a real valued differentiable function satisfying |f(x)- f(y)| \(\le\) \({(x-y)}^2\), x, y \(\in\) R and f(0) = 0, then f(1) equals-

Answer : 0

Question 3: The set of points where f(x) = \(x\over {1+|x|}\) is differentiable-

Answer : (\(-\infty, \infty\))

Question 4: If function f(x) is differentiable at x = a then \(\displaystyle{\lim_{x \to a}}\) \(x^2f(a)-a^2f(x)\over {x-a}\)

Answer : 2af(a) - \(a^2\)f'(a)

Question 5: Let f(x) = x|x|, g(x) = sinx and h(x) = (gof)(x). Then

Answer : h'(x) is continous at x = 0 but is not differentiable at x = 0

Question 6: Let f : R \(\rightarrow\) R be a function such that |f(x)| \(\le\) x^2, for all x \(\in\) R. Then, at x = 0, f is :

Answer : Continuous as well as differentiable

Question 7: If the function, g(x) = {k\(\sqrt{x+1}\), 0 \(\le\) x \(\le\) 3 and mx + 2, 3 < x \(\le\) 5 is differentiable, then value of k + m is-

Answer : 2

INVERSE TRIGONOMETRIC FUNCTION


Simplified Inverse trigonometric Functions


(a)   y = f(x) = \(sin^{-1}({2x\over {1+x^2}})\) = \(\begin{cases} 2tan^{-1}x, & \text{if}\ |x| \le 1 \\ \pi - 2tan^{-1}x, & \text{if}\ x > 1 \\ -(\pi + 2tan^{-1}x), & \text{if}\ x < -1 \end{cases}\)

(b)   y = f(x) = \(cos^{-1}({{1-x^2}\over {1+x^2}})\) = \(\begin{cases} 2tan^{-1}x, & \text{if}\ |x| \ge 0 \\ - 2tan^{-1}x, & \text{if}\ x < 0 \end{cases}\)

(c)   y = f(x) = \(tan^{-1}({2x\over {1-x^2}})\) = \(\begin{cases} 2tan^{-1}x, & \text{if}\ |x| < 1 \\ \pi + 2tan^{-1}x, & \text{if}\ x < -1 \\ -(\pi - 2tan^{-1}x), & \text{if}\ x > 1 \end{cases}\)

(d)   y = f(x) = \(sin^{-1}({3x - 4x^3})\) = \(\begin{cases} -(\pi + 3sin^{-1}x), & \text{if}\ -1 \le x \le {-1\over 2} \\ 3sin^{-1}x, & \text{if}\ {-1\over 2} \le x \le {1\over 2} \\ \pi - 3sin^{-1}x, & \text{if}\ {1\over 2} \le x \le 1 \end{cases}\)

(e)   y = f(x) = \(cos^{-1}({4x^3 - 3x})\) = \(\begin{cases} 3cos^{-1}x - 2\pi, & \text{if}\ -1 \le x \le {-1\over 2} \\ 2\pi - 3cos^{-1}x, & \text{if}\ {-1\over 2} \le x \le {1\over 2} \\ 3cos^{-1}x, & \text{if}\ {1\over 2} \le x \le 1 \end{cases}\)

(f)   y = f(x) = \(sin^{-1}({2x\sqrt{1-x^2}})\) = \(\begin{cases} -(\pi + 2sin^{-1}x), & \text{if}\ -1 \le x \le {-1\over \sqrt{2}} \\ 2sin^{-1}x, & \text{if}\ {-1\over \sqrt{2}} \le x \le {1\over \sqrt{2}} \\ \pi - 2sin^{-1}x, & \text{if}\ {1\over \sqrt{2}} \le x \le 1 \end{cases}\)

(f)   y = f(x) = \(cos^{-1}({2x^2-1})\) = \(\begin{cases} 2cos^{-1}x, & \text{if}\ 0 \le x \le 1 \\ 2\pi - 2cos^{-1}x, & \text{if}\ -1 \le x \le 0 \end{cases}\)


Example : Prove that : \(2tan^{-1}{1\over 2}\) + \(tan^{-1}{1\over 7}\) = \(tan^{-1}{31\over 17}\)

Solution : We have, \(2tan^{-1}{1\over 2}\) + \(tan^{-1}{1\over 7}\)

= \(2tan^{-1}({{2\times {1\over 2}\over {1-({1\over 2})^2}}})\) + \(tan^{-1}{1\over 7}\)       [\(\because\) \(2tan^{-1}x\) = \(tan^{-1}{2x\over {1-x^2}}\)]

\(tan^{-1}{4\over 3}\) + \(tan^{-1}{1\over 7}\) = \(tan^{-1}[{{4\over 3} + {1\over 7}\over {1 - {4\over 3}\times {1\over 7}}}]\) = \(tan^{-1}{31\over 17}\)


Equations involving Inverse trigonometric functions


Example : Prove that the equation \(2cos^{-1}x\) + \(sin^{-1}x\) = \(11\pi\over 6\) has no solution.

Solution : Given equation is \(2cos^{-1}x\) + \(sin^{-1}x\) = \(11\pi\over 6\)

\(\implies\) \(cos^{-1}x\) + (\(cos^{-1}x\) + \(sin^{-1}x\)) = \(11\pi\over 6\)

\(\implies\) \(cos^{-1}x\) + \(\pi\over 2\) = \(11\pi\over 6\)

\(\implies\) \(cos^{-1}x\) = \(4\pi\over 3\)

which is not possible as \(cos^{-1}x\) \(\in\) [0, \(\pi\)]. Hence no solution.




Inequations involving Inverse trigonometric functions


Example : Find the complete solution set of \(sin^{-1}(sin5)\) > \(x^2\) - 4x.

Solution : \(sin^{-1}(sin5)\) > \(x^2\) - 4x   \(\implies\)   \(sin^{-1}[sin(5-2\pi)]\) > \(x^2\) - 4x

\(\implies\) \(x^2\) - 4x < 5 - 2\(\pi\)   \(\implies\)   \(x^2\) - 4x + 2\(\pi\) - 5< 0

\(\implies\) 2 - \(\sqrt{9-2\pi}\) < x < 2 + \(\sqrt{9-2\pi}\)   \(\implies\)   x \(\in\) (2 - \(\sqrt{9-2\pi}\), 2 + \(\sqrt{9-2\pi}\))



Summation of Series


Example : Prove that : \(tan^{-1}({{c_1x-y}\over {c_1x+y}})\) + \(tan^{-1}({{c_2-c_1}\over {1+c_2c_1}})\) + \(tan^{-1}({{c_3-c_2}\over {1+c_3c_2}})\) + ...... + \(tan^{-1}({{c_n-c_{n-1}}\over {1+c_nc_{n-1}}})\) + \(tan^{-1}({1\over {c_n}})\) = \(tan^{-1}({x\over y})\)

Solution : L.H.S.

\(tan^{-1}({{c_1x-y}\over {c_1x+y}})\) + \(tan^{-1}({{c_2-c_1}\over {1+c_2c_1}})\) + \(tan^{-1}({{c_3-c_2}\over {1+c_3c_2}})\) + ...... + \(tan^{-1}({{c_n-c_{n-1}}\over {1+c_nc_{n-1}}})\) + \(tan^{-1}({1\over {c_n}})\)

= \(tan^{-1}({{{x\over y} - {1\over c_1}}\over {1+{x\over y}.{1\over c_1}}})\) + (\(tan^{-1}c_2\) - \(tan^{-1}c_1\)) + (\(tan^{-1}c_3\) - \(tan^{-1}c_2\)) + ..... + (\(tan^{-1}c_n\) - \(tan^{-1}c_{n-1}\)) + \(tan^{-1}({1\over {c_n}})\)

= \(tan^{-1}({x\over y})\) - \(tan^{-1}({1\over {c_1}})\) - \(tan^{-1}c_1\) + \(tan^{-1}c_n\) + \(tan^{-1}({1\over {c_n}})\)

= \(tan^{-1}({x\over y})\) - (\(cot^{-1}c_1\) + \(tan^{-1}c_1\)) + (\(tan^{-1}c_n\) + \(cot^{-1}c_n\))

= \(tan^{-1}({x\over y})\) - \(\pi\over 2\) + \(\pi\over 2\) = \(tan^{-1}({x\over y})\)
